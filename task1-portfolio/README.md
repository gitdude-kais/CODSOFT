# Kais Anjum — Developer Portfolio

A personal developer portfolio website built with pure HTML, CSS, and 
vanilla JavaScript as part of the CodSoft Web Development Internship (Task 1).

## 🌐 About This Project

This is a fully responsive, single-file portfolio website designed with a 
dark **night-cityscape** aesthetic. It showcases my skills, projects, and 
contact information in a visually rich, animated interface — with no 
frameworks or build tools required.

## ✨ Features

- Animated starfield canvas background
- Custom amber + teal cursor with smooth ring follow
- Sticky navbar with blur backdrop on scroll
- Scroll-reveal animations on all sections
- Animated skill progress bars
- Project cards with hover lift effects
- Testimonials section
- Resume download button
- Contact form UI
- Fully responsive (mobile-friendly)

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (Canvas API, IntersectionObserver)
- Google Fonts (Cormorant Garamond, Syne, JetBrains Mono)
