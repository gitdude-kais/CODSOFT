# MULTOR — Appointment Booking Landing Page

A clean, modern appointment scheduling landing page built with pure HTML 
and CSS as part of the CodSoft Web Development Internship (Task 2).

## 🌐 About This Project

MULTOR is a professional appointment booking landing page designed for 
service-based businesses. It features a two-column hero layout — a 
benefits section on the left and a booking form on the right — with a 
teal and dark charcoal colour theme.

## ✨ Features

- Fixed navbar with blur backdrop
- Animated hero tag with pulsing dot
- Two-column responsive hero layout
- Animated benefit cards (fade-up on load)
- Fully styled booking form with:
  - First & Last Name fields
  - Email & Phone fields
  - Appointment type dropdown
  - Schedule Now CTA button
- Social proof strip (ratings, client count, security badge)
- Fully responsive — collapses to single column on mobile

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Google Fonts (Playfair Display, DM Sans)
