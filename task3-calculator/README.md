# Calculator — Interactive Web Calculator

A fully functional, keyboard-supported calculator built with pure HTML, 
CSS, and vanilla JavaScript as part of the CodSoft Web Development 
Internship (Task 3).

## 🌐 About This Project

A sleek dark-themed calculator with a purple accent colour scheme. It 
supports all standard arithmetic operations, chained calculations, 
expression history display, and full keyboard input — all in a single 
self-contained HTML file with no dependencies.

## ✨ Features

- All basic operations — Add, Subtract, Multiply, Divide
- Percentage (%) and Sign toggle (+/−) buttons
- Expression history shown above the current value
- Chained calculations (e.g. 5 + 3 × 2 without pressing equals)
- Dynamic font resizing — display shrinks for long numbers
- Blinking cursor when actively typing
- Error shake animation on divide-by-zero
- Ripple animation on every button press
- Active operator highlight (purple glow on selected op)
- Full keyboard support:

| Key | Action |
|-----|--------|
| `0–9` | Digits |
| `+ - * /` | Operators |
| `Enter` or `=` | Equals |
| `.` or `,` | Decimal |
| `Backspace` | Delete last digit |
| `Escape` | Clear (AC) |
| `%` | Percent |

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, grid, keyframe animations)
- Vanilla JavaScript (event delegation, state machine logic)
- Google Fonts (Share Tech Mono, Syne)
